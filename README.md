## 📌 Project Overview
This project is a simple and responsive Invoice Generator built using HTML, CSS, and JavaScript. It demonstrates API integration by fetching real-time product data and dynamically displaying it on the webpage.

---

## 🎯 Purpose
The goal of this project is to:
- Learn how to integrate public APIs using fetch()
- Display dynamic data on a webpage
- Build a simple real-world application (Invoice Generator)

---

## 🌐 API Used
DummyJSON API  
https://dummyjson.com/products  

This API provides product data such as title, price, and category, which is used in the invoice generator.

---

## 🚀 Features

### ✅ Current Features
- Fetch product data using API
- Display products dynamically in dropdown
- Search products by name
- Filter products by category
- Select product and enter quantity
- Calculate subtotal, GST (18%), and total amount
- Print invoice
- Responsive design (works on mobile, tablet, desktop)

---

## 🔍 API Integration
The application uses the fetch() method in JavaScript to retrieve product data from the DummyJSON API. The fetched data is then displayed dynamically and used to calculate invoice values.

---

## 🌐 Additional API (Future Scope)
Tax Data API (APILayer)  
https://marketplace.apilayer.com/tax_data-api  

This API can be integrated in the future to calculate real-time tax rates based on region and improve the accuracy of invoice calculations.
